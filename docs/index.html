EDA Footprint Generator

Stop hand-drawing footprints. Describe a component once in YAML and generate the KiCad footprint, schematic symbol and 3D STEP model — validated against IPC-7351B before anything is written.

⬇ Install in KiCad View on GitHub

One YAML → three files, checked

Instead of drawing each pad, silk and courtyard by hand — where a 0.2 mm slip becomes a manufacturing defect — you describe the physical parameters and let the platform generate everything, standards-checked and reproducible.

🦶

Footprint

.kicad_mod — pads, silkscreen, courtyard and fab layers for KiCad 6/7/8.

🔣

Symbol

.kicad_sym — schematic symbol with named, typed pins.

🧊

3D model

.step — parametric solid via CadQuery. Imports straight into Fusion 360.

✅

IPC-7351B validated

Clearance, courtyard, annular ring and silk checks. Errors abort generation.

🔄

Multi-EDA export

KiCad .pretty, Eagle .lbr, Altium CSV and BOM.

📐

7 pad patterns

axial / radial / dual PTH, dual / quad SMD, custom and BGA.

Install via the KiCad Plugin & Content Manager

Add this repository to your KiCad PCM and install with one click.

Repository URL
  1. KiCad → Plugin and Content Manager → Manage…
  2. Click + and paste the URL above
  3. Find EDA Footprint Generator and click Install

…or use the CLI

# clone & set up (Windows)
git clone https://github.com/PedroWall-e/EDA-Footprint-Generator.git
cd EDA-Footprint-Generator
.\scripts\setup_ambiente.ps1

# generate footprint + symbol + 3D from a YAML
python cli.py gerar modulos_config/NE555_DIP8.yaml -o saida/
